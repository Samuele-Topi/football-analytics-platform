# Contributing to Football Analytics Platform

First off, thank you for considering contributing! It's people like you that make this platform a world-class tool for the football community.

## üöÄ How Can I Contribute?

### Reporting Bugs
*   Check the [Issues](https://github.com/user/repo/issues) to see if it has already been reported.
*   If not, open a new issue using the **Bug Report** template.
*   Include as many details as possible: steps to reproduce, expected behavior, and screenshots.

### Suggesting Features
*   Open a new issue using the **Feature Request** template.
*   Explain the "Why" behind the feature. How does it help a Data Engineer or a Full-Stack Developer?

### Code Contributions
1.  Fork the repository.
2.  Create a new branch (`git checkout -b feature/amazing-feature`).
3.  Follow the **[Developer Reminders](./REMINDERS.md)** for Git hygiene and versioning.
4.  Ensure your code follows the established style (see `docs/frontend` or `docs/backend`).
5.  Commit your changes (`git commit -m 'feat: add amazing feature'`).
6.  Push to the branch (`git push origin feature/amazing-feature`).
7.  Open a Pull Request.
